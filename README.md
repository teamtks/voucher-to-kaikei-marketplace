# voucher-to-yayoi-marketplace

証憑書類(領収書・請求書)から弥生会計インポートデータを作成するスキルを、
社内で共有するためのリポジトリです。

## 導入方法(各自のPCで1回だけ)

Claude Desktopアプリのチャット画面で `/plugin` と入力してもコマンドとして
認識されないため、**設定画面から操作**してください。

1. 左下(または歯車アイコン)から「設定」を開く
2. 左メニューの「Claude Code」→ さらにその中の「プラグイン」をクリック
3. 右上の「追加」→「マーケットプレイスを追加」
4. URL欄に `teamtks/voucher-to-kaikei-marketplace` と入力して「同期」
5. 一覧に表示された「証憑→弥生インポートデータ作成」の「+」ボタンをクリックしてインストール

これで、以降どのプロジェクトフォルダで作業していても、このスキルが自動的に
使えるようになります。

## 初回のみ必要な追加セットアップ(パソコンごとに1回)

このスキルは証憑画像の宛名を自動で黒塗りする処理のためにOCR(文字認識)の
プログラムを使います。これは**利用者が手作業で行う必要はありません** —
インストール後、Claude Codeのチャットで

```
voucher-to-yayoiスキルを使う準備をして
```

とお願いするだけで、Claude自身がフォルダの場所を特定し、必要なセットアップ
(このときだけインターネット接続が必要)を代わりに実行します。Pythonが
入っていないパソコンであっても、Claudeが自動でインストールします。

## 更新方法

このリポジトリの中身を更新した後、各自のPCで「設定」→「Claude Code」→
「プラグイン」の画面を開き、`voucher-to-yayoi-marketplace` の右にある
「…」メニューから再同期してください。

## 管理者向け: このマーケットプレイスに新しいスキルを追加する方法

今後、他の会計ソフト向けのスキルなどを追加する場合は、以下の手順で行います。

### 1. フォルダ構成

新しいスキルは `plugins/<スキル名>/` の下に、以下の形で追加します
(`voucher-to-yayoi` を参考にしてください)。

```
plugins/<スキル名>/
├── .claude-plugin/
│   └── plugin.json
└── skills/
    └── <スキル名>/
        └── SKILL.md (と必要なscripts等)
```

### 2. 【重要】ドットで始まるフォルダのアップロードに関する注意

`.claude-plugin` のような「.(ドット)」で始まるフォルダは、**GitHubのWeb画面で
ドラッグ&ドロップアップロードすると、ブラウザに隠しファイル扱いされて
アップロードから漏れてしまいます。** これは今回のセットアップ時に実際に
つまずいた点です。

回避策として、`.claude-plugin`フォルダの中身(`plugin.json`)は、
ドラッグ&ドロップではなく、GitHub画面の「Add file」→「Create new file」を使い、
ファイル名の欄に **フォルダ名を含めたパスをそのまま手入力** してください。

```
plugins/<スキル名>/.claude-plugin/plugin.json
```

このように入力すると、フォルダごと自動的に作成されます。

### 3. ルートの`marketplace.json`にも追加を忘れずに

リポジトリ直下の`.claude-plugin/marketplace.json`の`plugins`配列に、
新しいスキルの情報を1件追加してください(`voucher-to-yayoi`の記述を
コピーして書き換えるのが簡単です)。

### 4. 動作確認

自分のPCで「設定」→「Claude Code」→「プラグイン」→
`voucher-to-yayoi-marketplace`の「…」→再同期を行い、新しいプラグインが
一覧に表示され、インストールできることを確認してから、社内に案内してください。
